## Internship

### Research Centre Imarat (RCI), Hyderabad — Machine Learning Intern

**Project: AI-Based Telemetry Anomaly Detection System**

Developed an end-to-end anomaly detection pipeline for high-dimensional telemetry data using deep learning techniques. The goal was to identify abnormal patterns in telemetry signals to support predictive monitoring and improve system reliability.

| | |
|---|---|
| Model | LSTM Autoencoder for learning normal system behavior |
| Detection Method | Reconstruction error-based anomaly scoring |
| Optimization | NumPy vectorization for preprocessing efficiency |
| Evaluation | Anomaly detection metrics with pattern visualization |

**Key Contributions**
- Built a full anomaly detection pipeline on large-scale telemetry datasets using LSTM Autoencoders
- Performed data preprocessing, feature engineering, and normalization to improve model performance
- Optimized the preprocessing pipeline using NumPy vectorization for faster data processing
- Evaluated the model using reconstruction-error-based detection techniques
- Visualized anomaly patterns and analyzed model output to support reliable system monitoring
